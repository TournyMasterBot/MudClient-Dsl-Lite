# Install
* Install [nvm](https://github.com/coreybutler/nvm-windows)
* Install NodeJS v20 (nvm install 20)
* Use NodeJS v20 (nvm use 20)
* Install [Visual Studio Code](https://code.visualstudio.com/Download)

# Installation
* npm install [for first run]

## Play :: CLI
* npm run start

## Play :: Web
* npm run start:web
